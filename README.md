# 음식 추천 어플 Backend

오늘 먹을 메뉴를 추천해드리는 서비스의 백엔드입니다.

## Node & NPM Version

Node v16.13.2
NPM v8.3.1

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# build
$ npm run build

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
